> 基于nestjs的博客后台。目前实现jwt验证、注册登录、路由守卫、全局异常捕获、接口统一捕获。

### 🏠 [Homepage](https://user.starryskystar.space)

## Prerequisites

- npm >=6.14.11
- node >=v14.16.0

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```
